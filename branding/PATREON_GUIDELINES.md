# Patreon Brand Guidelines Reference

## Official Resources
- **Brand Page**: https://www.patreon.com/brand/
- **Assets (Google Drive)**: https://drive.google.com/drive/folders/13RZh5XzHE-336eYieOneojOirhNuowYl

## Logo Elements
- **Symbol**: The "P" icon
- **Wordmark**: "Patreon" text version
- **Primary Lockup**: Horizontal arrangement (preferred)
- **Alternative Lockup**: Vertical arrangement (space-constrained)

## Usage Rules

### Allowed
- Reference your creator status on Patreon
- Digital applications (websites, social media)
- Promote your creations and Patreon presence

### Prohibited
- Altering/distorting logos
- Merchandise use without agreement
- Merging with brand names
- Using logos as sentence elements

## Brand Colors
- **Patreon Coral**: #FF424D (primary brand color)
- **Black wordmark**: Use on light backgrounds
- **White wordmark**: Use on dark backgrounds

## Our Implementation
- Hero section button linking to https://www.patreon.com/c/Sopivasti
- Light mode: Black wordmark on light button
- Dark mode: White wordmark on dark button
- Uses official Patreon coral as accent
