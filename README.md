# Sopivasti - Mobile Apps

Simple portfolio site for my hobby mobile apps.

## Apps

- **Random Race** - Dinosaur racing decision maker
- **Finger Chooser** - Random finger selection tool
- **Digits** - Countdown numbers game solver

## Tech Stack

Static HTML/CSS/JS hosted on GitHub Pages.

No frameworks, no build tools, no analytics. Just plain files.

## Local Development

Open `index.html` in a browser. That's it.

## Deployment

Push to GitHub, Pages handles the rest.

---

Free apps. No ads. No tracking. Built as a hobby.
